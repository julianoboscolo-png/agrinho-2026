# agrinho-2026

**A página Agrinho 2026 é um site responsivo que apresenta o projeto educativo e cultural do Agrinho, destacando a conexão entre o campo e a cidade. Ela possui um cabeçalho colorido, menu de navegação, galeria de imagens com títulos e descrições, e um botão interativo em JavaScript que exibe mensagens. A página se adapta automaticamente a computadores, tablets e celulares, oferecendo uma experiência visual moderna e acessível para todos os usuários.**
